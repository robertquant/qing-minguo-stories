# 清末民国小说集

AI创作的清末民国题材短篇小说集，聚焦老北京旗人的生活变迁。

## 收录作品

1. **旗人往事** - 民国二十三年，两位老旗人在茶馆里追忆往昔
2. **茶馆里的八旗旧事** - 八旗子弟的黄昏岁月
3. **茶馆里的最后一出戏** - 戏曲艺人的时代悲歌
4. **落花时节** - 王府花园里的最后一场盛宴
5. **落日余晖** - 清末贵族的衰落与新生的抉择
6. **铁杆庄稼倒了** - 民国初年旗人的生存困境

## 创作背景

这些小说以1890-1940年代的北平为背景，聚焦于：

- **物质生活的变迁**：从铁杆庄稼到自谋生路，从豪宅大院到卖房典当
- **精神世界的失落**：对清朝的怀念、对现实的无奈、对未来的迷茫
- **身份认同的困境**：满洲故土沦陷、溥仪出走长春，旗人在家国之间的矛盾
- **生活细节的坚持**：遛鸟、茶馆、京片子，旗人文化在日常中的延续

## 创作说明

- 采用白描手法，语言简洁克制
- 以细节和对话推动叙事，避免说教与煽情
- 历史细节经过审查，力求还原时代风貌
