## 第一轮审查

### 1. 八旗制度相关

**问题发现**：

1. **镶蓝旗与步军营的搭配**：
   - 原文：德明是镶蓝旗人，祖上三代都在步军营当差
   - 核实：步军营属于八旗中的下五旗，镶蓝旗是下五旗之一，搭配正确 ✓

2. **善扑营与旗人**：
   - 原文：福海因为善扑营当差
   - 核实：善扑营是由旗人组成的摔跤部队，正确 ✓

3. **上三旗与护军营**：
   - 参考资料显示：护军营是上三旗（正黄、镶黄、正白）的专属，镶蓝旗不能进入护军营
   - 原文未涉及此问题 ✓

### 2. 旗人礼仪与语言

**问题发现**：

1. **打千儿**：
   - 原文：点头哈腰地打招呼
   - 问题："点头哈腰"是汉语成语，旗人见面应该说"打千儿"
   - **需修改**

2. **问候语**：
   - 原文："您吉祥！""托您的福，还消停着。"
   - 核实：这是典型的旗人问候语，正确 ✓

3. **旗人名与汉名**：
   - 原文：德明、福海等
   - 核实：清末旗人已有采用汉名的情况，正确 ✓

### 3. 饷银制度

**问题发现**：

1. **饷银金额**：
   - 原文未明确说明具体饷银金额，需要补充以增加真实感
   - 参考资料：马甲月饷四两银子，岁米四十斛

2. **铁杆庄稼**：
   - 原文使用"铁杆庄稼"一词
   - 核实：这是旗人对俸饷的俗称，正确 ✓

### 4. 历史时间线

**问题发现**：

1. **庚子年时间线**：
   - 原文：七月里，八国联军打进北京
   - 核实：八国联军1900年8月14日攻入北京，为农历七月二十日，正确 ✓

2. **辛亥年时间线**：
   - 原文：宣统三年，辛亥革命爆发
   - 核实：1911年10月10日武昌起义，为农历八月十九日
   - 问题：原文说"十月里，武昌起义的消息传到北京"
   - **需修改**：应为"九月里"（农历八月的消息到北京大约需要半个月）

3. **民国时间线**：
   - 原文：民国元年德寿中学毕业
   - 时间线：德寿光绪十三年（1887年）生，民国元年（1912年）25岁
   - 问题：这个年龄中学毕业偏大，但考虑到当时教育水平，可以接受
