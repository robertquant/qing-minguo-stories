# 祖传的翡翠扳指

冬日的北平，寒风卷着枯叶在胡同里打转。正阳门外的打磨厂胡同，一处三进的小四合院里，六十八岁的瓜尔佳·德贵正坐在暖阁的炕上，手里摩挲着那枚祖传的翡翠扳指。

这扳指是祖上从龙入关时得的赏赐，通体温润，翠色欲滴。德贵他爷爷在道光年间做过参领，那时候瓜尔佳家在镶黄旗也算是有头有脸的人家。可如今，大清国亡了十几年了，旗人的铁杆庄稼早没了，德贵只能靠变卖家什度日。

"阿玛，今儿个煤球又涨价了。"儿媳秀珍端着热茶进来，脸上带着愁容，"咱家的煤怕是撑不到开春了。"

德贵没吭声，只是把扳指攥得更紧了些。这扳指，有人出过三百大洋，他没卖。那时候还指望着朝廷能回来，指望着旗人的体面能回来。可这一年年的，民国了，军阀打仗了，日子越过越艰难。

街坊们都说德贵是死脑筋。隔壁那钮祜禄家的福海，早把祖宅卖了，一家人搬去天津做买卖，听说混得还不错。可德贵舍不得，这宅子是祖产，卖了，就真什么都没了。

腊月二十三，小年夜。德贵的孙子小宝从外面跑进来，兴冲冲地说："爷爷，琉璃厂那边有个铺子收翡翠，说是给高价！"

德贵心里一紧。他看着小宝那张稚嫩的脸，又看了看炕桌上剩的半个窝头。小宝今年才十二，本该是读书的年纪，可学堂的学费都欠了半年了。

第二天天还没亮，德贵就起来了。他从柜底翻出一件还算体面的长袍，把扳指揣进怀里，拄着拐杖出了门。

琉璃厂的荣宝斋，掌柜的是个精明的南方人。他拿放大镜仔细看了看扳指，说："老先生，这是好东西，可惜了，这时候节，没人敢出高价。我给您一百二十块，您看如何？"

德贵的手抖了抖。一百二十块，够买多少煤球，够交多少学费，够让他一家老小吃上几顿饱饭。可这扳指，是他最后的体面了。

"一百五。"德贵咬着牙说。

掌柜的犹豫了一下，点了点头。

德贵揣着银元往回走，路过前门楼子的时候，他忍不住回头看了一眼。那城墙还在，可这北平，这天下，早就不是他年轻时那个样了。

回到家，德贵把银元往炕上一倒，秀珍和小宝都愣住了。

"阿玛，您……"秀珍的眼圈红了。

"拿着吧。"德贵的声音有些沙哑，"先把煤买了，再把小宝的学费交了。剩下的，留作过年的花销。"

小宝扑过来抱着德贵的腿："爷爷，我不上学了，我不让您卖扳指！"

德贵摸着孙子的头，苦笑了一下。这孩子，还不知道，没了扳指，他们家还得继续过日子。可好歹，这个冬天，不用挨冻了。

晚上，德贵躺在床上，辗转难眠。他想起小时候，爷爷戴着这扳指，教他骑马射箭。那时候，旗人还是旗人，这北平还是京城。如今，一切都变了。

可转念一想，日子不就是这么过的吗？大清国亡了，可这四九城还在，这胡同还在，这一家老小还在。扳指没了，可人还在，只要人在，这天塌下来，也能顶着。

德贵翻了个身，沉沉睡去。梦里，他又回到了年轻时候，穿着笔挺的马褂，骑着高头大马，在什刹海边上溜达。扳指在手上闪闪发光，阳光照在身上，暖洋洋的。

第二天一早，小宝兴冲冲地跑进来："爷爷！外头下雪了！"

德贵起身，推开窗户。果然，漫天雪花飘着，整个北平城银装素裹。

"走，"德贵笑了，"陪爷爷去什刹海转转。"

祖孙俩走在雪地里，小宝问："爷爷，咱家以前是不是大户人家？"

德贵点点头："是，也不是。你爷爷我啊，年轻时候也不知道天高地厚。后来才明白，啥叫过日子。"

"那爷爷，咱家现在是不是穷了？"

德贵蹲下身，看着孙子："小宝啊，穷不穷的，不看家里有多少银子，看的是咱心里头有没有股劲儿。只要这股劲儿在，咱瓜尔佳家，就永远是体面人。"

雪越下越大，祖孙俩的身影渐渐消失在胡同尽头。

远处，什刹海的冰面上，有人在滑冰，有人在抽陀螺。这北平城，这日子，就这么一天天地过着。有人走了，有人来了，有人富了，有人穷了。可这城还在，这些人还在，这烟火气还在。

德贵紧了紧长袍，心想，没了扳指，可这人还在，家还在。明天，还得继续过呢。
