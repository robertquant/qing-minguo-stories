# 旗人最后的体面

民国二十三年，北平的秋天来得格外早。

德胜门内大街的镶黄旗老宅院里，六十八岁的瓜尔佳·瑞祥正对着镜子梳理他那条精心护理的花白辫子。虽然大清早已亡了二十多年，街上早已看不见几条辫子，可瑞祥爷还是固执地留着，他说这是祖宗的规矩，不能丢。

"爷，今儿个鸟市还去吗？"老仆人德顺在院子里问。德顺本是瑞祥家的包衣，民国后也没了主仆名分，可还是跟着瑞祥，不要工钱，就图口饭吃。

"去，怎能不去？"瑞祥爷戴上天青色的瓜皮帽，拿起那只在鸟市上淘来的百灵笼子，"今儿个纪昌那小子说他弄了只好画眉，我去瞧瞧。"

瑞祥爷提着鸟笼，踱着方步出了大门。街上黄包车来来往往，叫卖声此起彼伏。"卖冰糖葫芦——""驴打滚儿——"瑞祥爷目不斜视，可裤兜里那几个铜板，他心里清楚得很。

前清时候，瓜尔佳家在德胜门外有三顷好地，城里还有三进的大院子。庚子年八国联军进来，烧了一部分；辛亥后，又"自愿"捐了一部分给新政府，说是支持共和。到如今，就剩这不足两进的破院落，还有祖上留下来的一点体面。

鸟市在西四牌楼，瑞祥爷走了小半个时辰才到。其实他可以叫辆洋车，可那得花钱。旗人的体面，有时候就体现在这走路的姿态上——再穷，腰杆也得直。

"哟，瑞祥爷来了！"纪昌是个三十来岁的汉人，祖上是贩鸟的，到他这一辈还干这个。他凑上来，压低声音："爷，您看今儿个，那画眉——"他指了指东边一个摊位。

瑞祥爷顺着瞧去，那摊位上坐着个四十来岁的男人，穿一身半旧的青布长衫，手里转着两个核桃，面前摆着只画眉笼子。画眉鸣声清脆，确实不错。

"这鸟多少钱？"瑞祥爷问。

"十二块大洋。"那人说。

瑞祥爷心里一惊。十二块大洋，够他和德顺吃一个月的。可旗人的规矩，相中了的东西不能让人看轻。他咳嗽一声，伸手接过笼子，仔细端详。

"鸟是好鸟，"瑞祥爷说，"就是鸟缰绳扣得紧了些，怕是野路子。"他说的是鸟架上的绳扣。

那男人笑了："爷是行家。这鸟是我上个月从西山那边收的，还没完全驯服。"

两人攀谈起来。瑞祥爷发现这人姓佟，也是旗人，镶白旗的，祖上当过佐领，如今在西单摆摊卖字画为生。民国了，同是天涯沦落人。

"十二块太贵了，"瑞祥爷说，"六块，我拿走。"

佟姓男人犹豫了一下："爷，这价我进价都不够。"

"八块，不能再多了。"瑞祥爷掏出怀表看了看，那表早停了，就是个样子货，"我家中还有几本前清的字帖，改日给您送去，贴补贴补。"

佟姓男人眼中一亮："是吗？我正收这些。那成交。"

瑞祥爷掏出八块大洋，一块一块数给对方。那手有点抖，这些钱本是他从当铺里把亡妻的金镯子赎回来攒下的。

回家的路上，瑞祥爷提着画眉笼子，脚步轻快了许多。他想起了二十多年前，大清刚亡那会儿，旗人们聚在茶馆里痛骂袁世凯，骂着骂着就抱头痛哭。那是何等的没落。如今呢？该活还得活，该提鸟还得提鸟。这是旗人的体面，活一口气。

到家时，德顺已经做好了饭——棒子面窝头，一碟咸菜，一碗稀粥。

"爷，买着了？"德顺问。

"买着了。"瑞祥爷把鸟笼挂在廊下，"好画眉，叫得好听。"

德顺没说话，默默盛了粥。他知道爷又要从当铺里赎东西了，可他不说。旗人的体面，有时候就是不说破。

瑞祥爷坐在八仙桌前，看着窗外的秋光。院子里那棵老枣树叶子黄了一半，风吹过来，簌簌地响。这树是他祖父种的，那时瓜尔佳家还风光着呢。

他端起粥碗，喝了一口，又放下。

"德顺啊。"

"爷。"

"明天去把那对玉扳指当了，"瑞祥爷说，"去庆云楼订个席面，请佟爷来家里坐坐。"

"爷，那扳指是祖上传下来的……"

"我知道。"瑞祥爷沉默了一会儿，"可旗人不能没朋友，不能没了这点人脉。扳指当了，以后还能赎，朋友没了，这日子还怎么过？"

德顺低下头，没再说话。

窗外，画眉叫了两声，清脆悦耳。瑞祥爷笑了。他想，大清亡了，旗人还在；好日子没了，体面还在。只要这口气还在，什么都还在。

秋风从德胜门城楼上吹过来，带着北平特有的干燥和尘土味。瑞祥爷闭上眼睛，仿佛又听见了什刹海边的鸽哨声，那是从前八旗兵遛鸟放鸽时的响器，如今还偶尔能听到。

民国了，可他还是瓜尔佳家的瑞祥爷。这，变不了。
